#  🗓️ News Feed Project
## ❔프로젝트 정보
- Spring, JPA, 3 Layer Architecture를 활용한 'News Feed' 협업 프로젝트 입니다.<br>
- 기본적인 유저 관리, 뉴스피드 게시물 및 댓글 관리, 좋아요, 팔로우/팔로잉 기능을 구현하였습니다.<br>
- **개발기간**: 24.12.20 ~ 24.12.27

### 🔷 주요 기능

- 유저 관리, 게시글 관리, 팔로워 관리 기능

### 🔷 부가 기능

- 댓글, 좋아요, 기간별 게시물 조회 기능
